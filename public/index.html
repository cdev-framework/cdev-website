Cdev

The Cdev Framework

A dynamic framework that enables scalability and managability for serverless projects of any size. By using abstractions created in high level programming languages, Cdev reduces boilerplate configuration and promotes composability. Cdev helps developers focus on the parts of their application that make them unique.
